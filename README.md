# 🛒 Nishu Mart - Premium E-Commerce Platform\n\nA fully responsive, premium Indian e-commerce marketplace similar to Flipkart and Amazon with modern UI/UX, ultra-fast performance, mobile-first design, and advanced customer-friendly features.\n\n## ✨ Features\n\n### 🎨 Design\n- Premium blue + yellow + white color scheme\n- Glassmorphism effects\n- Dark/Light mode support\n- Mobile-first responsive design\n- Professional typography\n\n### 🛍️ Core Features\n- Auto-rotating hero banner\n- 12 product categories\n- Flash sale with countdown timer\n- Shopping cart with persistence\n- Product ratings and reviews\n- Search functionality\n\n### 📱 Mobile Optimization\n- Touch-friendly interfaces\n- App-like experience\n- Fast loading speeds\n- Responsive grid layouts\n\n## 🚀 Tech Stack\n\n- **Frontend**: Next.js 14, React 18, TypeScript\n- **Styling**: Tailwind CSS\n- **State Management**: Zustand\n- **Backend**: Supabase (PostgreSQL)\n- **Icons**: Lucide React\n- **UI**: Custom components\n\n## 📦 Installation\n\n### Prerequisites\n- Node.js 18+\n- npm or yarn\n\n### Setup\n\n1. **Clone the repository**\n   ```bash\n   git clone https://github.com/nikhilsingh48576-stack/nishu-mart.git\n   cd nishu-mart\n   ```\n\n2. **Install dependencies**\n   ```bash\n   npm install\n   ```\n\n3. **Run development server**\n   ```bash\n   npm run dev\n   ```\n\n4. **Open browser**\n   ```\n   http://localhost:3000\n   ```\n\n## 📂 Project Structure\n\n```\nnishu-mart/\n├── app/                    # Next.js pages & layouts\n├── components/             # React components\n├── lib/                    # Utilities & constants\n├── stores/                 # Zustand stores\n├── styles/                 # Global CSS\n├── types/                  # TypeScript definitions\n└── supabase/               # Database migrations\n```\n\n## 🎯 Key Components\n\n- **Header** - Navigation, search, theme toggle\n- **Hero Banner** - Auto-rotating carousel\n- **Categories** - 12 shopping categories\n- **Flash Sale** - Live countdown timer\n- **Footer** - Links and information\n\n## 📊 Database Tables\n\n- users\n- products\n- categories\n- orders\n- cart_items\n- And more...\n\n## 🚀 Deployment\n\n### Deploy to Vercel\n```bash\nnpm install -g vercel\nvercel\n```\n\n## 📝 License\n\nMIT License -
